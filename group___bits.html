int sdl::most_significant_bit_index ( uint32_t  x)

Get the index of the most significant bit.

Result is undefined when called with 0. This operation can also be stated as "count leading zeroes" and "log base 2".

Returns
Index of the most significant bit, or -1 if the value is 0.
